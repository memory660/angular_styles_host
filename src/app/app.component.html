:host

- à utiliser pour définir le style de l'élément hôte dans votre feuille de style
- pratique pour supprimer les éléments HTML inutiles (comme les conteneurs)
- utilisez-le avec des variables CSS pour le rendre personnalisable depuis l'extérieur du composant.

HostBinding

- permet d'attribuer dynamiquement des classes/styles/attributs à l'élément hôte
- permet de supprimer les doublons de code dans les CSS et les HTML
- tirer parti des attributs de données pour rendre l'interface utilisateur dynamique.

:host-context

- permet de définir un style différent pour l'élément hôte en fonction du contexte d'utilisation
- convient pour adapter l'interface utilisateur à d'autres endroits de l'application sans dupliquer les composants
